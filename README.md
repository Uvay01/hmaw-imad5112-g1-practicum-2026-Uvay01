## About the Project
Campsite Commander is an Android application developed for outdoor enthusiasts to manage their camping inventory. The app stores camping gear and food supplies, categorizes them, and tracks the quantities using parallel arrays. 

## Features
* **Splash Screen:** A welcoming screen with a camp-themed logo.
* **Base Camp (Main Menu):** Displays the total number of items packed and provides navigation.
* **Detailed View (Packing List):** Displays all the stored gear, categories, quantities, and specific notes in a scrollable list.

## How-To Guide
1. **Starting the App:** Launch the app on your Android emulator or device. The Splash Screen will appear for 3 seconds before navigating to the Base Camp menu.
2. **Checking Total Items:** On the Base Camp menu, you will see the "Total Items Packed" text. This calculates the sum of all quantities currently stored in the system.
3. **Viewing the Packing List:** Click the "View Packing List" button.
4. **Reading the List:** The Detailed View screen will display every item currently in your inventory, showing its Name, Category, Quantity, and any Notes.
5. **Returning to Base Camp:** Click the "Back to Base" button at the bottom of the Detailed View screen to return to the main menu.
